# 🍕 Real-Time Full-Stack Pizza Ordering Application

A highly performant, production-grade pizza ordering and tracking application built with the **Next.js 15 App Router**, **TypeScript**, **Zustand**, and **Supabase**. 

This repository was engineered with a strict focus on modularity, clean architectural boundaries, robust input validation, and real-time state synchronization via PostgreSQL logical replication event streams.
